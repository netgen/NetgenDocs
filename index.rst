Welcome to Netgen Docs
======================

This page lists all docs available for Netgen products.

Netgen Layouts
--------------

`Netgen Layouts`_ is an easy and flexible way of managing all page layouts on
your website. Use it to manage and maintain the website layout structure in a
slick and agile but at the same time extensible way.

Netgen Content Browser
----------------------

`Netgen Content Browser`_ is a Symfony bundle that provides an interface
which selects items from any kind of backend and returns the IDs of
selected items back to the calling code.

eZ Platform Site API
--------------------

`eZ Platform Site API`_ enables translating your eZ Platform content model to a
website in clean, natural and streamlined way, often without even writing PHP.
It makes the process of building websites easier, faster and less error prone.

eZ Platform Search Extra
------------------------

`eZ Platform Search Extra`_ provides additional features for eZ Platform search.
